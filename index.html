Intro

Hi! I'm Anthony. A software engineer with experience in JavaScript, utilizing React & Redux for frontend development & Ruby on Rails and PostgreSQL for backend. With a background in customer trend analysis, music production and graphic design, I have a passion for creating and creative problem solving. Breaking down complex problems into small digestible solutions, finding patterns and searching for improvement in performance are skills I've honed through out each of my previous experiences.

I developed a passion for programming after writing small scripts in After Effects to automate animations. I quickly realized coding was the perfect blend of logic, problem solving and creativity I've been looking for.

artHouse

artHouse is a full-stack, single-page art sharing/social app inspired by Instagram using Ruby on Rails, JavaScript, React.js, Redux and Postgresql.

It Optimizes aggregation of user feed and suggested artists leveraging custom SQL queries and ActiveRecord methods & associations to avoid N + 1 queries. I Created reusable React components such as Follow and Like buttons for more maintainable and DRY code. Having a passion for design and aesthetics I also leverage HTML5 and CSS to replicate Instagrams sleek, user friendly design and image constraints.

Ghoul-Alley

Ghoul-Alley is an original 2d browser game using original artwork.

This game utilizes HTML5 Canvas for custom sprite rendering & collision detection. Sprite sheet animation, as well as tracking key controls for responsive gameplay is implemented through VanillaJS. I also Leverage object-oriented design inheritance to maintain DRY code.

RubyRM

RubyRM is a lite Object-Relational Mapping (ORM) library modeled after Active Record which creates relationships to easily retrieve records from the database without writing SQL.
